# Game Space Project

Game Space is my first more-advanced project that I decided to share with the world. It's a video game library built using the RAWG API. I failed a lot and learned a ton of lessons from creating this app. Below, I list the most important information about everything I used during the development.

## Technologies I used

#### Framework: React

As I said, this is my first project of such scale. That's why I wanted to use as few external libraries as possible in order to sharpen my React skills to the best level I could. The only major library I used was React Router, and that's it.

#### Styling: CSS Modules using Sass

The path I took at the start was to go with basic Sass; only in the middle of development did I realize that the files were truly unreadable, so I changed the tactic and switched to modules, but kept the preprocessor.

## Images and Icon Sources

All the images I taken to build the app came from the game developers themselves. So all the credit goes to their artist. On the other hand Icon I used were all taken from [SVG Repo](https://www.svgrepo.com/). To cut, model and optimize the Images i used [Resize Img](https://www.iloveimg.com/resize-image) as well as [Squoosh](https://squoosh.app/).

## Inspiration

The main and only inspiration I draw from is the offical RAWG library. The same guys I have taken the API from. Hovewer, I still think that the majority of the app was made with my own vision as a guide.
